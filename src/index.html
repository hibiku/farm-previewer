Farm Previewer

How to use

Config
Select the level, dimensions and monster capacity of your farm.
Check "Use Aufheben effect" if the Aufheben monster's effect applies to your farm.
Mode: Inspect
Within a cell's properties, any bonuses from road network effects applied to the building/decor that occupies it are marked in (+) parentheses.
Mode: Decorate
Select the type and name of the object to be inserted into the grid.
Check "Autofill roads" to automatically fill free cells in the grid with roads.
Grid
When in "Inspect" mode, click on a cell in the grid to see its properties.
When in "Decorate" mode, click on a free cell in the grid to insert the selected object at its location.
When a cell is selected, it will be outlined in red.
Click the "Remove" button to remove the object that occupies the selected cell.
Click the "Reset" button to remove all objects that occupy cells in the grid.
