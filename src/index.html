Farm Previewer

How to use

Config
Select the level, dimensions and monster slots of your farm.
Check "Use Aufheben effect" if the Aufheben monster's effect applies to your farm.
Mode: Inspect
Within a tile's properties, any bonuses from road network effects (and/or the Aufheben monster's effect, for buildings) applied to the object that occupies it are marked in (+) parentheses.
Click on a tile in the grid to view its properties. The tile being inspected will be outlined in red, and if it is occupied by an object, any other tiles occupied by that object will also be outlined in red.
Mode: Decorate
Select the type and name of the object to be inserted into the grid.
Click on a tile in the grid to insert the selected object at its location as the root. The newly occupied tiles will be colored accordingly (as shown in the grid legend) and outlined in red.
Check "Autofill roads" to automatically fill blank tiles with road objects, and vice-versa.
Grid
Select a preset from local storage, and then click the "Load"/"Delete" button to load/delete the preset from local storage. (Warning: Loading a preset will override the current config settings/grid objects.)
Click the "Create" button to create a preset using the current config settings/grid objects, after entering a title for the preset. (Warning: Any presets created while running this app in your browser's private browsing mode will not persist after closing all private windows.)
Click the "Remove" button to remove the object that occupies the tiles(s) outlined in red.
Click the "Clear" button to remove all objects that occupy tiles in the grid.
Stats
Hover over a data point in the "Production timeline" graph to reveal a tooltip displaying the data at that point.
Click on an item in the legend to show/hide the corresponding dataset. (Note: The items in the legend are the titles of any saved presets, in addition to "Current," which refers to the config settings/grid objects currently in use.)
